Contexte
Space Shooter
Projet réalisé en 2024

"Space Shooter" est un jeu de tir en 2D où le joueur contrôle un vaisseau pour détruire des ennemis qui descendent de l'écran. Le but est d'accumuler des points en évitant que les ennemis n'atteignent le bas de l'écran. Le joueur utilise les touches fléchées pour se déplacer et la touche espace pour tirer. Les ennemis apparaissent à intervalles réguliers avec des vitesses aléatoires, ajoutant un défi. Le score est affiché en haut de l'écran et mis à jour en temps réel. Si un ennemi atteint le bas de l'écran, le jeu se termine et le score final est affiché.

La structure
Space Shooter

Création du jeux : Le jeu est créé avec HTML pour la structure, CSS pour le style, et JavaScript pour la logique. Le canvas est utilisé pour le rendu graphique, et les fonctions JavaScript gèrent les mouvements, les collisions et les interactions utilisateur. Les écouteurs d'événements permettent de contrôler le vaisseau et de tirer. Le jeu est lancé en appelant les fonctions nécessaires pour initialiser et maintenir la boucle principale.
